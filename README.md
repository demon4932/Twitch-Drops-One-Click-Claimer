# Twitch Drops One Click Claimer

Twitch Drops One Click Claimer is a lightweight browser extension that makes claiming your earned Twitch drops fast and effortless. Instead of clicking through each drop manually, this extension lets you claim everything with one single click.

### **Chrome and Brave (Load Unpacked Installation)**

**Download the extension folder as a ZIP:**
[Download ZIP](https://github.com/demon4932/Twitch-Drops-One-Click-Claimer/raw/refs/heads/main/TwitchAutoClaim.zip)

**Installation steps:**

1. Download or clone the extension folder to your computer
2. Open your browser and go to:
   **chrome://extensions**
3. Enable **Developer Mode** (top right)
4. Click **Load unpacked**
5. Select the folder you extracted from the ZIP
6. The extension will now appear in your browser
